# -My-Python-Learning-Journey
这里面放的程序一个是上课时打过无数遍的计算BMI的程序代码，可以根据输入的身高与体重输出对应的BMI数值与对应的状态。
另一个是第一次作业中写的解决鸡兔同笼问题的程序代码，可以识别输入的数据是否合理，然后输出对应的鸡和兔子的数量。

## 学习心得与规划
之前也用过github来获取一些游戏脚本来帮我清理明日方舟等游戏的日常委托，不过从来没有试图去了解过这些脚本背后运行的机制。
这次作业的过程让我了解了如何在这个网站上传自己的代码，也让我更乐于在这里寻找些有意思的程序来扩展视野。
未来我也会把课上见到的我能熟练掌握的代码上传到这里，记录自己的成长。
